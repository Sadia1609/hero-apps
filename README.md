## Description

Hero Apps is a responsive app marketplace web application where users can explore popular apps, view detailed ratings and reviews, install apps using localStorage, and manage their installed applications with a smooth user experience.

## Core Features

Fully responsive design for all devices

Dynamic app listing with live search

App details page with review analytics (charts)

App installation & uninstall using localStorage

Toast notifications for user actions

Sorting apps by download count

Custom error and loading states

Route reload support after deployment

## How to Run the Project Locally

Follow these steps to run Hero Apps on your local machine:

# Step 1: Clone the repository
git clone https://github.com/your-username/hero-apps.git

# Step 2: Navigate to the project folder
cd hero-apps

# Step 3: Install dependencies
npm install

# Step 4: Start the development server
npm run dev
